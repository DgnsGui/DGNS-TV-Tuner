## üì∫ Overview

**DGNS TV Tuner** is an open-source Lens project designed for **Snap Spectacles**.  
It provides a lightweight and extensible framework to watch live HLS (M3U8) streams directly in AR.

The project is intended as:
- a **technical reference**
- a **creative playground**
- and a **starting point** for custom Spectacles TV / media experiences

Users are responsible for providing and using **legally authorized streams only**.

---

## üöÄ Features

- **WebView**  
  Load and play your own HLS stream URLs.

- **Example Demo Channels**  
  A small curated set of authorized and official streams for testing.

- **Simple & Extensible Architecture**  
  Easy to modify, extend, or integrate into other Spectacles projects.

- **Spectacles-Ready UI**  
  Designed for hands-free AR usage and spatial interaction.
